Description

The Goon RDA (by Tobeco) is a 22mm RDA that uses an innovative “bridge clamp” design unrivalled by any other design in the industry.

Installing massive and intricate builds or the simplest coils has never been easier with the bridge clamp design of the Goon.

These clamps allow for a massive amount of room to install coils without being restricted to post holes that only accept a certain amount of wire.

The Goon's bridge clamps are gold plated for maximum conductivity as well as a flashy looking build deck, to add to the gold plated clamps comes the copper 510 contact to make power travel from mod to coils in a pinch.

Integrated with PEEK insulators and a Delrin 1/2 wide bore drip tip, the Good is built with top notch materials to improve it's longevity.

Truly innovative and unrivaled, the Goon proposes a particular way to build in a whole new package.

Wash the unit before use. This is to rid of machine oil or any other left over from production process. Washing the Tank / Atomizer is something that should be done for any new device
Please Note: The bottom feeding screw plug is preinstalled and should be removed using the included 0.9mm hex key for squonk mod compatibility
